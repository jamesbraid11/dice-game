# Hackathon Dice Game


![Dice image](assets/Dice_Red.png)

### Outline
This is a Dice Game where you will roll a dice against the computer for 1-3 rounds. Who will be victorious?

### How to run the game:
- Fork repo
- In the terminal, run:
~~~
npm i
~~~
- In the terminal, run:
~~~
node index.js
~~~
- Follow the instructions in the terminal and enjoy the game!
